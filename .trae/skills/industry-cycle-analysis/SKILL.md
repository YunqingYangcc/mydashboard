# 产业周期推演技能 (Industry Cycle Analysis Skill)

## 技能概述
本技能用于判断一个行业或公司在技术趋势中的周期位置，从而指导买入、持有或卖出决策。
核心思想：**用产业思维做提前布局，赚认知和耐心的钱。**
适用场景：分析 AI、新能源、生物科技等强技术驱动的产业。

## 核心理念
> **炒股的本质**：用合理的价格，买入一个未来能持续创造超额价值、且你能大致看清其发展路径的行业或公司。
> **完整策略**：提前布局（在逻辑验证早期、价格合理时进场） + 持续跟踪逻辑不被破坏 + 在市场过度定价时逐步退出。

## ⚠️ 强制要求：搜索最新数据
**每次推演前，必须先搜索该标的的最新数据，包括但不限于：**
- 最新季度财报数据（营收、利润、毛利率、指引）
- 最新行业动态（竞争格局变化、技术突破、政策法规）
- 最新市场情绪（分析师评级、机构持仓变化、媒体热度）
- 最新催化剂事件（产品发布、监管审批、大客户签约）

**严禁使用过时数据做推演。如果搜索不到最新数据，必须明确告知用户数据时效性不足。**

## 所需知识框架（四层）
1. **底层规律（骨）**：技术成熟度曲线、创新扩散曲线、产业价值链微笑曲线。
2. **产业追踪（肉）**：需求信号、供给信号、催化剂辨别。
3. **交叉验证（将知识串联）**：四问清单。
4. **排除噪音（反面知识）**：四个不等。

## 推演流程（口诀）
> **虚实分开上下游，曲线上把阶段瞅。三看一问扫信号，四个不等记心头。**

### 第一步：选赛道 —— 虚实分开上下游
- **虚实**：判断是数字世界AI（虚拟）还是物理世界AI（现实）。
  - 数字AI：周期短，兑现快，易找发令枪。
  - 物理AI：周期长，落地难，发令枪远。
- **上下游**：判断在价值链的位置。
  - 上游（卖铲人）：确定性最高，先受益。
  - 中游（模型/平台）：格局未定，赌赢家。
  - 下游（应用）：弹性最大，需看是否跨过鸿沟。

### 第二步：定阶段 —— 曲线上把阶段瞅
- **技术成熟度曲线**：定位市场情绪和估值阶段。
  - 技术触发期 → 太早，不买。
  - 期望膨胀期 → 市场发疯，是卖点。
  - 泡沫谷底期 → 研究谁活下来。
  - 复苏验证期 → **黄金买入区**。
  - 实质生产期 → 持有，等下一次疯狂。
- **创新扩散曲线**：找鸿沟跨越的"必买时刻"。
  - 产品在极客和早期使用者手里 → 尚早。
  - 普通大众开始"非用不可" → **黄金买入点**。
  - 最好在业绩刚证明自己时上车，放弃最早一段利润换取确定性。

### 第三步：扫信号 —— 三看一问，缺一不动手
- **一看需求**：付费用户是否加速？企业有无"非买不可"的理由？保守大客户是否开始采购？
- **二看供给**：技术路线收敛了吗？龙头份额集中了吗？毛利率趋势？利润在向哪个环节转移？
- **三看催化剂**：
  - 启动型（发令枪）：降本增效实锤、法规落地、巨头降低应用门槛。
  - 尾声型（离场钟）：非主营公司蹭概念、媒体全民狂欢、算多年后市场空间。
- **一问（交叉验证）**：
  - 跨过泡沫谷底了吗？
  - 找到跨越鸿沟的必买理由了吗？
  - 数据确认拐点了吗？（付费增速、毛利率等）
  - 当前价值链上确定性最高、利润最肥的环节在哪？
  四个问题全部明确指向，才能动手。

### 第四步：排噪音 —— 四个不等
- 融资额高 ≠ 周期来了
- 大佬站台 ≠ 周期来了
- 自己/朋友在用 ≠ 周期来了
- 研报说万亿市场 ≠ 周期来了

## 使用提醒
- 该技能用于**辅助决策**，不代替财务分析和估值分析。
- 产业周期判断是**概率游戏**，即使全部信号绿灯，也要做好黑天鹅预案。
- 定期回测，检查信号是否持续成立，逻辑是否被破坏。
- **必须搜索最新数据**，严禁用过时信息做推演。
