# 智谱AI 产业周期推演

## 三维速查卡
- 价值链位置：中游（平台服务商——以MaaS为核心）
- 实现形式：虚拟（纯虚拟AI）
- 周期阶段判定：技术成熟度曲线——**期望膨胀期 → 泡沫谷底期（临界跨越中，正面临从预期到兑现的考验）**；IPO后股价暴涨12倍+估值破万亿港元+营收暴增300%但净亏损同步扩大至41亿以上，市场已在高位显著透支未来数年预期；创新扩散曲线——处于 **"鸿沟跨越早期"** ，已在独立大模型领域站稳脚跟，但面临大型云厂商MaaS平台的围堵和"管道化"风险
- 关键观察信号（需求/供给/催化剂）：①需求：API业务ARR达17亿元，过去一年提升60倍；GLM Coding Plan上线20天订阅用户突破40万，调用量逆势涨价后暴涨400%；②供给：GLM-5系列快速迭代（5代/半年），GLM-5.2即将开源；AutoGLM 2.0发布，对标GPT-5 Agent Mode；③催化剂：港股上市股价暴涨12倍（启动型）；回A计划150亿募资（启动型）；API逆势涨价83%后调用量不降反增400%（最关键的"必买时刻"信号）
- 核心风险：①股价短期暴涨超12倍，PS高达210倍，已严重透支未来数年预期，任何业绩不及预期都可能导致剧烈回撤；②营收极度依赖政府与国企的私有化部署，客户集中度极高，复购不确定性大；③亏损持续扩大——2026年预期净亏损超40亿，且预计亏损状态将持续至2028年；④"管道化"风险被低估——大型云厂商（阿里云、火山引擎、腾讯云）正加速整合MaaS平台，可能大幅压缩独立大模型厂商的议价空间
- 行动建议：**不碰/观望**——当前股价已脱离基本面，市场处于典型的"预期膨胀"阶段。PS 210倍对应一家年亏损超40亿且仍处高投入期的公司，风险收益比极度不对称。等待两大信号：①出现行业性"洗牌"信号，泡沫被充分消化；②亏损收窄趋势确认，且MaaS平台议价能力得到实质性验证。


## 推演过程

### 第一步：选赛道 —— 虚实分开上下游

智谱AI属于典型的 **"虚拟 + 中游平台服务商"** 定位。

**虚实判断**：智谱AI是纯虚拟AI标的，主营业务为大模型API调用服务（MaaS平台）、私有化部署方案及AI编程/智能体产品。不涉及物理制造，周期特性为虚拟属性——受益于AI应用端的快速爆发，弹性大、波动剧烈，商业化兑现节奏较快。

**上下游判断**：智谱AI定位在**中游**（平台服务商），提供MaaS（Model as a Service）一站式平台，通过API向企业和开发者提供大模型调用服务。与上游"卖铲人"（提供GPU算力等基础设施）不同，智谱AI的价值在于模型能力本身——通过不断迭代基座模型的技术领先性，形成客户粘性和议价能力。正因如此，智谱的独特卡位在于**不依赖自有庞大云基础设施**，而是通过技术专注力和模型迭代速度，在国内外市场形成差异化竞争优势。智谱被市场定位为"国家队+产业侧主力"，客户以机构客户为主，To B权重高，在国内独立通用大模型开发商中位列第一，按2024年收入计市场份额达6.6%，在所有通用大模型开发商中位列第二。

但中游平台服务商的内生挑战在于：大型云厂商（阿里云百炼、火山引擎方舟、腾讯云混元等）通过"模型超市"模式，将多个模型整合到同一平台，部分客户可能会从直接调用独立模型厂商的API，转向通过云厂商一站式采购。如果独立模型厂商不能建立足够深的技术护城河，将面临显著的"管道化"风险——即API调用本身被云厂商的聚合平台架空，失去议价能力。

### 第二步：定阶段 —— 曲线上把阶段瞅

智谱AI当前处于 **"技术成熟度曲线"从期望膨胀期向泡沫谷底期跨越的临界窗口**，同时也面临"跨过鸿沟"的关键挑战。

**【技术成熟度曲线定位】**

**技术触发期（2023年-2024年）** ：ChatGPT引爆AI大模型浪潮，智谱凭借GLM系列技术积累脱颖而出，聚焦"Code + Agent"双轮驱动战略，在国内大模型竞争中逐步站稳独立厂商第一梯队位置。

**期望膨胀期（2025年初-2026年5月，全面进入）** ：港股IPO引爆市场热情。2026年1月8日，智谱以"全球大模型第一股"身份登陆港交所，发行市值528亿港元。随后股价一路暴涨至1412港元，市值突破6500亿港元，较发行价上涨超12倍。API业务ARR约17亿元，过去一年提升60倍，平台注册企业及用户突破400万，在全球218个国家地区提供服务。2026年3月，GLM Coding Plan（Claw Plan）上线仅两天订阅用户即破10万，上线20天订阅用户突破40万，验证了智能体长链路任务的巨大商业空间。

**泡沫谷底期（2026年Q2，预期将进入）** ：当前阶段核心矛盾：市值已突破6500亿港元，但2026年预期收入仅约30亿元，净亏损预期超40亿元。PS估值高达约210倍，对应一家年亏损数十亿仍处高投入期的企业，泡沫化程度极高。3月，智谱API逆势涨价83%，但调用量反而增长约400%，向市场传递了"客户为能力付费、而非为低价付费"的信号。

**复苏验证期（目前尚未进入）** ：要实现从泡沫谷底向复苏验证期的跨越，需要依赖A股二次上市融资落地、GLM-5等新模型持续商业化验证、Agent场景规模化落地等实质性突破。

**【创新扩散曲线的"鸿沟"判断】**

智谱正处于"鸿沟跨越"的关键挑战期：

- **鸿沟正在跨越的积极信号**：
  1. **API涨价后调用量反增**：逆势涨价83%，调用量增长约400%，显示客户正从"尝鲜"转向"刚需"，为真正的必买时刻提供了有力验证。
  2. **Agent产品落地加速**：2026年3月推出"AutoGLM沉思"智能体，可实现从数据收集、模型构建到方案落地的全流程自主化。5月发布AutoGLM 2.0，对标OpenAI GPT-5 Agent Mode。6月推出AutoGLM 2.0，为AI配置云端专属智能体手机/电脑，支持AI自主干活且不占用用户本地设备。
  3. **生态合作加速拓展**：与滴滴合作探索出行Agent场景落地，与找钢网共建B2B全流程通用型AI Agent，与洲明科技成立合资公司布局AI智能终端，参与岚图汽车车载智能体支付场景。
  4. **港股股价暴涨12倍**，成为"大模型赛道第一股"，大幅领先国内其他独立大模型厂商的融资和估值水位，获得了跨越鸿沟所需的关键弹药。

- **鸿沟仍然需要面对的问题**：
  1. **"管道化"风险正在加速放大**：阿里云百炼、火山引擎方舟、腾讯云混元等大型云厂商正在构建"模型超市"生态——火山引擎在大模型中标榜单中已位列行业第二、国内MaaS平台调用量49.5%市场垄断性第一。客户可能从直接购买智谱API，转向通过云厂商同时调用多个模型。当客户发现云厂商的MaaS平台集成度更高、综合成本更低时，独立模型厂商将加速被"挤压"。
  2. **规模化落地的压力巨大**：截至2026年3月，智谱API ARR约17亿元，虽然增速（60x）可观，但绝对规模仍远低于第一梯队云厂商的MaaS平台收入。当前股价定价了"年收入200亿+"级别的预期，而实际商业化节奏与市场预期之间存在巨大裂痕。

### 第三步：扫信号 —— 三看一问

#### 一看需求：付费用户加速？企业必买理由？大客户采购？

需求信号**强劲，呈现"量价齐升"的特征，但基数仍偏小**：

1. **API业务呈现"量价齐升"——最重要的需求信号**：
   - 2026年3月，智谱在发布GLM-5的同时，将核心模型API价格上调约83%。在价格大幅上涨后，调用量却逆势增长了约400%。
   - 截至2026年3月，API业务ARR已实现约17亿元，过去12个月增长了60倍。毛利率则从2024年的3.4%跃升至18.9%，虽然绝对值不算高，但增幅够大。

2. **AI编程订阅数据超预期**：
   - GLM Coding Plan上线两日后迅速售罄，被市场解读为"国产AI编程的付费热度高涨"的关键信号。Claw Plan上线仅两天订阅用户即破10万，20天突破40万，验证了智能体长链路任务的巨大商业空间。

3. **平台生态规模持续扩大**：
   - 平台注册企业及用户突破400万，服务全球超过218个国家及地区。
   - 客户名单已包括三六零、滴滴、找钢网、洲明科技、极豆科技、岚图汽车等各行业头部企业。

4. **需求端的结构性风险——客户集中度高**：
   - 招股书显示，智谱2024年收入高度依赖面向政府、国企的私有化部署项目，客户集中度高且复购存在不确定性。
   - 当前MaaS平台在公有云MaaS市场的营收份额，仍远低于火山引擎、阿里云等云厂商。智谱面临的一个残酷现实是：客户买你的API，却可能通过云厂商的MaaS来调用。

#### 二看供给：技术收敛？龙头份额集中？利润转移？

供给信号**技术迭代迅猛，但竞争格局日益复杂**：

1. **模型迭代速度极快——技术不收敛，反而在加速分化**：
   - 2025年以来，智谱加速迭代，先后发布GLM-4.5、GLM-4.6、GLM-4.7、GLM-5和GLM-5.1，每一代都在国际最具影响力的评测中刷新纪录——全球开源模型第一。
   - 2026年6月13日，智谱宣布GLM-5.2将面向GLM Coding Plan全量用户开放，API将于下周上线并开源，遵循MIT协议。这意味着智谱正在走"开源+闭源"双轨路线的技术差异化路径，短期内有望维持在国内独立模型厂商中的技术优势。

2. **MaaS市场竞争格局——存量分化，头部效应加剧**：
   - 在公有云MaaS市场中，按营收口径，火山引擎占据40%以上的市场份额，阿里云、百度智能云、智谱、移动云位列前五，火山引擎以断层优势领跑。
   - 按MaaS Token调用量计算，火山引擎以49.5%的市场份额断层领先，智谱排名第七，差距悬殊。
   - **核心矛盾**：智谱的收入（6.6%的行业份额）在独立模型厂商中名列前茅，但在云厂商的MaaS整合面前仍在被强势挤压。当云厂商的MaaS成为企业应用部署的默认入口，独立模型厂商就会丧失议价权。

3. **利润转移尚未到来，亏损持续扩大**：
   - 2026年预期净亏损将超过40亿元。
   - 营收与亏损同步高速扩张——营收预计将从2025年的约7亿元暴涨至2026年的约30亿元，但净亏损也将从2025年的约23亿元进一步扩大到2026年的约41亿元。多家券商预测2026-2028年净亏损合计约110亿元，直到2028年后才可能接近盈亏平衡。

#### 三看催化剂：启动型（发令枪）vs 尾声型（离场钟）

**启动型催化剂（已点燃，持续发酵中）** ：

1. **港股上市股价暴涨超12倍**——启动型最强信号，市值已突破6500亿港元。市场用真金白银押注智谱将成为"中国的OpenAI"。但当前市值透支了至少未来3-5年的增长预期。

2. **回A计划，再募150亿**——启动型第二关键信号。6月1日，智谱董事会通过建议A股科创板发行决议，拟募集资金150亿元，其中120亿元用于AI通用基座大模型，20亿元用于MaaS一站式服务平台。150亿资金若顺利到账，将是智谱实现跨越式发展的关键弹药。

3. **API逆势涨价83%，调用量暴涨400%**——最关键的启动型需求验证信号。这证明定价权正在形成，客户愿意为更高的模型能力支付更高价格，是"鸿沟跨越"的核心验证点之一。

4. **AutoGLM 2.0发布，对标OpenAI GPT-5**——2026年5月16日，智谱宣布完成5亿美元融资，同日发布AutoGLM 2.0 Agent平台。与此同时，GLM-5.1入选《时代》周刊"2026年最具影响力的10家AI公司"。

**尾声型催化剂（需持续观察风险）** ：

1. **PS估值的泡沫化**：多家券商预测2026年营收均值约30亿元。当前市值约6500亿港元，对应PS约210倍，这在大模型公司中虽不算异常，但对标海外同行（如OpenAI、Anthropic均未上市，缺乏可比标的），市场信心已过度依赖"叙事"而非"数据"。

2. **高管流失的警示信号**：2025年上半年，智谱多位高管离职，包括应用AI与合作副总裁李惠子、首席战略官张阔、副总裁曲滕、COO张帆和CFO彭建标等。高管团队的不稳定，对处于高投入扩张期的创业公司是重大隐患。

#### 一问：四个交叉验证

1. **产业链上下游是否同步景气？** ⚠️ 分化明显
   - 上游：GPU算力持续紧张，智谱API价格涨价后仍供不应求
   - 中游：独立模型厂商（智谱、MiniMax、月之暗面）与云厂商MaaS（火山引擎、阿里云）的竞争日益激烈
   - 下游：Agent应用爆发，GLM Coding Plan和Claw Plan短期内订阅用户突破数十万
   - **分化信号**：云厂商的MaaS平台正凭借生态绑定优势，挤压独立模型厂商的市场空间

2. **市场份额是否集中在龙头？** ⚠️ 在MaaS层高度集中，但独立模型厂商整合趋势尚不明朗
   - 在公有云MaaS市场（营收口径），火山引擎占据40%以上的市场份额，集中度极高
   - 在独立模型厂商阵营中，智谱以6.6%市场份额处于领先位置，但相对于云厂商的体量仍较小
   - **深层挑战**：智谱虽然在独立模型厂商中"最强"，但独立模型厂商整体，正在被云厂商的MaaS生态系统挤压

3. **毛利率是否稳定向上？** ✅ 快速提升中
   - 从2024年的3.4%跃升至18.9%（截至2026年Q1）。API涨价83%后调用量不降反增，预计未来几个季度毛利率还将继续提升。但绝对值与海外头部模型厂商仍有差距。

4. **盈利质量是否改善？** ❌ 尚无明显改善
   - 营收高速增长（+300%+），但净亏损同步扩大，预期2026年净亏损约41亿元。自由现金流持续为负。
   - 多家券商预测2026-2028年将连续亏损，累计亏损约110亿元。

### 第四步：排噪音 —— 四个不等

**❌ 融资额高 ≠ 周期来了**
- 港股IPO募资约49亿港元+回A计划再募150亿元，融资总额约160亿元人民币，在大模型赛道中处于领先水平。但**"拿到钱"和"商业模式跑通"是两回事**。OpenAI、Anthropic等海外同行也曾大量融资，但利润率仍承压明显。巨额融资确认了市场方向，但不能替代对公司基本面能力的验证。

**✅ 创始人大佬站台 ≠ 周期来了**
- ⚠️ 混合噪音：智谱首席科学家、清华大学计算机系教授唐杰频繁露面阐述技术路线，对市场预期有提振作用。但管理层在IPO后发布积极技术预期与乐观商业化指引时，处于"既是科学的又是市场的"角色定位，存在一部分过度乐观的风险，需与真实业绩数据对比验证。

**⚠️ 自己/朋友在用 ≠ 周期来了**
- ⚠️ 噪音程度中高：很多个人开发者和学生正在使用GLM Coding Plan（近期20天超40万用户），这种个体使用体验容易让投资者误判商业化真实规模。C端用户的涌入对MaaS的B端核心商业化规模影响较小。

**✅ 研报说万亿市场 ≠ 周期来了**
- 华安、华源、开源、国信、招银国际等多家券商近期密集给予"买入/增持/优于大市"评级，逻辑围绕AI Agent爆发、模型能力跃升、商业化加速。智谱当前PS达约210倍，即便用最乐观的2028年203亿元营收计算，PS也高达约32倍。

**⚠️ 最危险的噪音：将"API涨价后调用量反增"过度线性外推，认为智谱已建立不可撼动的议价能力，低估了大型云厂商通过MaaS平台整合模型服务带来的"管道化"风险。**
