# 资本市场认知

> 面向8年经验全栈工程师+会计背景。核心问题：技术趋势≠股价走势，为什么？好公司≠好股票，为什么？

## 一、市值逻辑：好公司≠好股票

核心公式：**股价 = 盈利 × 估值倍数（PE）**

好公司可能有高盈利增长，但如果市场已经把未来3-5年的增长定价在当前股价中，你买入时赚的是零。

| 情景 | 盈利增长 | PE变化 | 股价表现 |
|------|---------|--------|---------|
| 增长超预期+PE扩张 | +30% | 30x→40x | +69% |
| 增长符合预期 | +30% | 30x→30x | +30% |
| 增长符合预期+PE收缩 | +30% | 30x→22x | -4.7% |
| 增长低于预期+PE收缩 | +15% | 30x→22x | -15% |

**程序员类比**：技术趋势是"需求"，股价是"已发布版本"。如果需求已经在发布说明里写清楚了，发布当天不会带来惊喜。只有超出release notes的预期差才能驱动股价。

**当前英伟达案例**：
- FY2026营收+65%（$2159亿），但增速从FY2025的+114%减速
- 股价$225（2026.5），Forward PE ~24x
- 市场已定价：持续增长+利润率维持
- 风险：增速进一步放缓→PE收缩→即使盈利增长，股价可能不涨甚至下跌

## 二、PE扩张/收缩的驱动因素

PE不是随机数，它由三个变量决定：

**PE = f(盈利增速预期, 利率水平, 风险偏好)**

| 因素 | PE扩张（利好） | PE收缩（利空） |
|------|--------------|--------------|
| 盈利增速预期 | 加速或超预期 | 减速或低于预期 |
| 利率 | 下降或维持低位 | 上升或通胀超预期 |
| 风险偏好 | 流动性宽松、VIX低位 | 地缘风险、流动性收紧 |
| 行业地位 | 垄断加强、护城河加深 | 竞争加剧、份额流失 |

**当前数据**：
- 标普500 Forward PE：20.9x（5年均值19.9x，10年均值18.9x）→ 高于均值但非极端
- IT板块Forward PE：20.0x（从2024年高点的32.0x大幅压缩）→ 已回调至接近20年均值18.6x
- 半导体远期PE：37.3x，5年预期增长率22.9%，PEG 2.13 → 偏贵
- AI板块PEG：0.6（25x Forward PE / 38.5%盈利增速）→ 成长性消化估值

**核心逻辑**：PE扩张需要盈利增速>利率增速。当前10年期美债收益率~4.3%，NVDA盈利增速虽高但正在减速。PE大幅扩张的窗口正在关闭。

## 三、CapEx周期：上游公司的命运由下游决定

这是理解AI供应链最关键的框架：

```
云厂商CapEx决策 → GPU/芯片订单 → 上游营收 → 上游估值
     ↑ 核心变量        ↑ 传导         ↑ 结果      ✓/✗
```

| 阶段 | 云厂商行为 | 上游影响 |
|------|-----------|---------|
| 扩张期 | CapEx YoY +60-77% | NVDA/光模块营收爆发，PE扩张 |
| 维持期 | CapEx增速放缓至10-20% | 上游营收增速骤降，PE收缩 |
| 收缩期 | CapEx绝对值下降 | 上游营收下滑，戴维斯双杀 |

**当前矛盾**：
- 华尔街预测超大规模云CapEx增速~10%
- 黄仁勋预测40%
- 这个差距终将缩小——而收敛方向决定NVDA股价方向

**自由现金流警告**：
- 亚马逊2026年FCF预计为负（-$170至-$280亿）
- Alphabet FCF预计下降近90%至$82亿
- 超大规模云2026年发债$1750亿（前5年年均$280亿的6倍）
- **当FCF转负且债务飙升时，CapEx减速只是时间问题**

## 四、流动性：牛市的真正本质

**牛市 = 流动性扩张 + 盈利增长，不是单纯的"经济好"**

| 指标 | 当前状态 | 对市场含义 |
|------|---------|-----------|
| 美联储资产负债表 | $6.71万亿（2026.5） | QT暂停，维持模式 |
| 美国M2 | ~$21.5万亿，YoY +3-5% | 温和扩张，支持风险资产 |
| ONRRP余额 | <$1000亿（从$2.5万亿峰值） | 流动性缓冲接近耗尽 |
| 银行准备金 | $3.0-3.2万亿 | 处于"充足"下限 |

**M2与SPX的相关性**：+0.72（12个月远期回报），这是最强的单一预测指标之一。

**关键阈值**：
- M2 YoY > +5%：强Risk-On环境
- M2 YoY 0-5%：中性偏支持
- M2 YoY < 0%：风险资产大概率下跌（历史上只出现2次，都伴随大跌）

**当前判断**：流动性环境中性偏支持。M2温和增长+QT暂停+利率高位但不再上升。这是"盈利驱动"而非"估值扩张"的市场——选股比选时更重要。

## 五、宏观周期：利率→估值→股价的传导

```
美联储利率决策 → 国债收益率 → 股权风险溢价 → PE → 股价
     ↑               ↑              ↑          ↑      ↑
  通胀/就业        债券供给       风险偏好    盈利预期   最终结果
```

| 情景 | 10Y收益率 | 对科技股PE影响 | 对NVDA影响 |
|------|----------|-------------|-----------|
| 降息+软着陆 | 3.5-4.0% | PE扩张10-20% | 强利好 |
| 维持+通胀温和 | 4.0-4.5% | PE维持 | 中性 |
| 加息+通胀超预期 | >4.5% | PE收缩15-25% | 利空 |
| 衰退+紧急降息 | <3.5% | 先跌后涨 | 短空长多 |

**摩根士丹利观点**：2026年EPS $339（+23% YoY），盈利驱动而非估值扩张。Forward PE维持20.5x（从当前21.2x轻微收缩）。

## 六、从程序员视角理解：技术趋势≠股价走势

| 认知偏差 | 程序员思维 | 投资本质 |
|---------|-----------|---------|
| "技术好=该买" | 技术评估 | 估值评估：好技术可能已定价 |
| "趋势明确=安全" | 确定性偏好 | 市场已反映确定性，风险在于不确定 |
| "增长快=空间大" | 线性外推 | 增长终将减速，PE先于盈利见顶 |
| "公司厉害=不会跌" | 公司视角 | 股价是边际定价，30%增长<50%预期=跌 |
| "长期持有=安全" | 代码不坏不需要改 | 估值过高时持有=承担均值回归风险 |

**英伟达的教训**：
- FY2026营收+65%（$2159亿）→ 仍然是超高增长
- 但增速从+114%降至+65% → 增速拐点已现
- 市场定价的是"增速维持"，而非"绝对值增长"
- 类比：代码性能从100ms优化到50ms是巨大进步，但用户预期的是20ms

## 七、实操判断框架：定价合理vs过度定价

### 定价评估清单

| 评估维度 | 合理定价信号 | 过度定价信号 |
|---------|-----------|-----------|
| Forward PE vs 5年均值 | ≤均值+1σ | >均值+2σ |
| PEG比率 | <1.5 | >2.5 |
| CAPE | <30 | >40 |
| 盈利增速vs PE | 增速>PE倍数 | PE倍数>增速 |
| FCF/CapEx | >0.5 | <0（转负） |
| 债务融资占比 | <30% CapEx | >60% CapEx |
| 机构持仓集中度 | Mag 7<25% | Mag 7>35% |

**当前评估**：

| 指标 | 当前值 | 判断 |
|------|--------|------|
| 标普Forward PE | 20.9x | 偏高（75%分位），但非极端 |
| CAPE | 39.5-40.11 | 极端偏高（98%分位，历史第二高） |
| IT板块Forward PE | 20.0x | 回调至合理区间（从32x压缩） |
| Mag 7权重 | 34-35% | 极端偏高（=2000年峰值） |
| M2 YoY | +3-5% | 中性偏支持 |
| 超大规模云FCF | 多家转负 | 警告信号 |

**综合判断**：整体市场处于"偏贵但不疯狂"区间。AI基础设施链估值已从2024年高点大幅回调（IT板块PE从32x→20x），但CAPE和集中度指标仍指向系统性风险。**精选个股比指数投资更安全。**
