## 各主题学习大纲

### 主题1：财务报表深度分析框架（P0）

**为什么要学**：你有会计背景，这是最大的认知优势，需要一个可复用的分析checklist

**学习内容**：
- [ ] 三表联动分析：利润表↔资产负债表↔现金流量表的勾稽关系
- [ ] 盈利质量判断：经营现金流 vs 净利润，应收账款异常，存货风险
- [ ] 杜邦分析：ROE = 净利率 × 资产周转率 × 权益乘数
- [ ] 财务造假识别：10个红旗信号（獐子岛、康美、瑞幸案例）
- [ ] 同行业对比模板：如何用5分钟筛选出行业内龙头 vs 尾部
- [ ] 实战：用框架分析英伟达、风华高科、寒武纪的财报

**产出**：一个可直接使用的财务分析checklist + 三家公司的分析案例

### 主题2：资产配置理论与实践（P0）

**为什么要学**：从"选标的"升级到"配组合"，用数学优化风险收益比

**学习内容**：
- [ ] 现代投资组合理论（MPT）：有效前沿、夏普比率、最优组合
- [ ] 相关性矩阵：为什么分散化是唯一的"免费午餐"
- [ ] 大类资产相关性：股债负相关何时失效？（2022年股债双杀案例）
- [ ] 风险平价策略：桥水全天候的底层逻辑
- [ ] 再平衡的数学：为什么定期再平衡能提升收益
- [ ] 实战：用理论优化你当前的90万+25万配置方案

**产出**：资产配置理论框架 + 优化后的配置方案

### 主题3：量化投资与自动化（P0）

**为什么要学**：你是程序员，这是天然优势，把投资规则代码化减少情绪干扰

**学习内容**：
- [ ] 回测框架：用Python/backtrader回测你的投资策略
- [ ] 因子投资：价值、成长、动量、质量因子的定义和应用
- [ ] 信号监控自动化：PE分位预警、10Y UST阈值提醒、持仓占比监控
- [ ] 组合优化算法：均值-方差优化、Black-Litterman模型
- [ ] 数据源搭建：Tushare/AKShare/Yahoo Finance API
- [ ] 实战：在你的dashboard里实现3个自动化信号

**产出**：量化框架文档 + dashboard自动化功能升级

### 主题4：期权策略实战指南（P1）

**为什么要学**：你已理解期权机制，现在要从"理解"到"使用"

**学习内容**：
- [ ] 期权定价基础：Black-Scholes、隐含波动率、Greeks
- [ ] Covered Call：持有NVDA时卖出看涨期权，降低持仓成本
- [ ] Protective Put：用看跌期权给NVDA买保险
- [ ] Cash-Secured Put：在想买入的价位收权利金等待建仓
- [ ] 铁鹰策略：在震荡市场中获利
- [ ] 实战：为你的26股NVDA设计一个期权增收方案

**产出**：期权策略框架 + NVDA期权操作方案

### 主题5：宏观经济周期框架（P1）

**为什么要学**：补齐"现在处于经济周期哪个位置"的判断能力

**学习内容**：
- [ ] 美林时钟：经济周期四阶段与最优资产配置
- [ ] 领先/同步/滞后指标：PMI、消费者信心、失业率、CPI的信号含义
- [ ] 中国经济结构：房地产、出口、消费、基建的驱动逻辑
- [ ] 康波周期：50-60年长波，AI是否是新一轮康波起点？
- [ ] 实战：判断当前全球经济处于周期的哪个阶段

**产出**：宏观周期判断框架 + 当前周期定位

### 主题6：债券与固收投资基础（P2）

**为什么要学**：债券是组合的"减震器"，高利率环境下尤其重要

**学习内容**：
- [ ] 债券定价原理：价格与收益率反向、久期、凸性
- [ ] 中国债券市场：国债、城投债、信用债的风险收益特征
- [ ] 债券ETF：511010、511260等标的的选品逻辑
- [ ] 股债平衡：60/40组合的历史表现和失效场景
- [ ] 实战：评估是否需要在你的组合中加入债券ETF

**产出**：债券投资框架 + 组合优化建议

---

### 主题7：投资税务规划（P2）

**为什么要学**：每年省几千到几万的"隐形收益"

**学习内容**：
- [ ] A股资本利得税：目前免征，但政策可能变化
- [ ] 港股红利税：20%的系统性规划
- [ ] 美股税务：股息预扣税10%、W-8BEN表格、资本利得税
- [ ] 跨市场税务优化：哪些标的放A股、港股、美股效率最高
- [ ] 实战：为你当前持仓做一次税务效率评估

**产出**：税务规划框架 + 持仓税务优化方案
